RSA - Удаленный запуск приложений, используя протокол TCP для связи. В первую очередь программа создавалась для отработки связи двух компьютеров по протоколу TCP, но я думаю, может кто-нибудь найдет ей ещё какое-нибудь применение.
Так как программа написана ИСКЛЮЧИТЕЛЬНО при помощи WinAPI-функций, то соответственно размер сервера получился ~6Кб и клиента ~5Кб. 

Клиент(TCP.EXE):

IP Address: надо ввести IP адрес машины для удаленного управления в формате "000.000.000.000".

Message: название программы для запуска.

Сервер:

Server.exe (~6Кб)

Не имеет окна и не виден в "панели задач".
